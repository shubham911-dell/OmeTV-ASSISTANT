# OmeTV Assistant

Smart country filtering, keyboard shortcuts, and session analytics for OmeTV.

## Features

### ⚡ Auto Skip

Automatically skip countries you don't want to see.

* Block specific countries
* Near-instant skipping
* Multiple speed modes
* Lightweight performance

### 🌍 Only Countries Mode

Only connect with people from selected countries.

Example:

* Philippines
* Japan
* Canada

All other countries are automatically skipped.

### ⌨ Keyboard Shortcuts

Navigate OmeTV faster using customizable shortcuts.

Default shortcuts:

* Next → Right Arrow
* Stop → Left Arrow

### 📊 Session Analytics

Track your OmeTV activity.

Includes:

* People Talked
* Skipped By You
* Skipped By Others
* Talking Time
* Top Chats
* Country Statistics

### 📈 History

View up to 7 days of activity history.

Track:

* Daily conversations
* Daily skips
* Country activity
* Chat duration

## Privacy

All settings and statistics are stored locally in your browser.

OmeTV Assistant does not collect personal information and does not sell user data.

## Permissions

The extension only runs on supported OmeTV pages to provide filtering, shortcut, and analytics features.

## Installation

### Manual Installation

1. Download the extension source.
2. Open Chrome.
3. Go to:

chrome://extensions

4. Enable Developer Mode.
5. Click Load Unpacked.
6. Select the extension folder.
