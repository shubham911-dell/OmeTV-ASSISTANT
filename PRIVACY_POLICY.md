# Privacy Policy

Last Updated: June 2026

## Introduction

OmeTV Assistant is a browser extension designed to improve the OmeTV experience through country filtering, keyboard shortcuts, and session analytics.

This Privacy Policy explains how the extension handles information.

## Information We Collect

OmeTV Assistant does not collect, sell, rent, or share personal information.

The extension does not require:

* User registration
* User accounts
* Email addresses
* Personal profiles

## Local Storage

The extension stores certain settings locally within your browser.

Examples include:

* Blocked countries
* Allowed countries
* Keyboard shortcut preferences
* Session statistics
* History statistics
* Extension settings

These settings remain on your device and are not transmitted to the developer.

## Website Interaction

OmeTV Assistant operates only on supported OmeTV pages to provide extension functionality.

This includes:

* Country filtering
* Shortcut controls
* Analytics features
* User interface enhancements

The extension does not interact with unrelated websites.

## Data Sharing

OmeTV Assistant does not:

* Sell user data
* Share personal information
* Transfer browsing data to third parties

## Analytics

All analytics displayed by the extension are generated locally on the user's device.

No analytics data is transmitted to external servers.

## Security

Settings and preferences are stored using browser-provided storage systems designed for browser extensions.

## Children's Privacy

OmeTV Assistant does not knowingly collect personal information from children.

## Changes to This Policy

This Privacy Policy may be updated periodically to reflect improvements or changes to the extension.

Any updates will be reflected by revising the "Last Updated" date above.
